__このBotの動作させるには__
```
TOKEN=XXX(Botログイン用token)
ADMIN=XXX(製作者ID , /admin_ 許可やレポート時の招待リンク送信のため)
SERVER_ID=XXX(レポートを送信するためのサーバーID)
CLIENT_ID=XXX(BotのclientID)
CHANNEL_ID=XXX(レポートを送信するためのチャンネルID)
SUPPORT_SERVER_INVITE=XXX(サポートサーバーへの招待リンク , 別途コードの招待リンク書き換えも必要)
SPOTIFY_CLIENT_ID=XXX(SpotifyのclientID)
SPOTIFY_CLIENT_SECRET=XXX(SpotifyのSECRETID)
YOUTUBE_API_KEY=XXX(YouTubeのAPIキー)
LOG=XXX(ログを送信するためのWebhookURL , Discordのみ)
```
